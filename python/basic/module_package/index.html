    Modules and Packages¶

    Module¶

    мүҪкІҢ мғқк°Ғн•ҳл©ҙ Python code лЎң кө¬м„ұлҗң file .pyмһҗмІҙлҘј к°ҖлҰ¬нӮЁлӢӨ.

    • ModuleмқҖ importлҗ  л•Ң к°Ғк°Ғ кі мң мқҳ namespaceлҘј к°Җм§ҖлҜҖлЎң, ModuleмқҖ мқјмў…мқҳ Namespaceмқҙкё°лҸ„ н•Ё.
    • Pythonм—җм„ң moduleмқҖ нҷ•мһҘмһҗк°Җ .pyмқё нҢҢмқјмқ„ к°ҖлҘҙнӮҙ ( python codeлҘј лӮҙмҡ©мңјлЎң к°Җм§ҖлҠ” file ).
    • лӘЁл“  .py fileл“ӨмқҖ pythonм—җм„ң moduleлЎңм„ң мӮ¬мҡ©к°ҖлҠҘн•Ё.

    Pythonм—җм„ң ModuleмқҖ codeмқҳ мһ¬мӮ¬мҡ©мқ„ мң„н•ҙ мӮ¬мҡ© лҗҳл©°,
    мқјл°ҳм ҒмңјлЎң moduleл“Өмқҙ лӘЁм—¬ н•ҳлӮҳмқҳ н”„лЎңк·ёлһЁмқҙ лҗңлӢӨ.

    codeмқҳ мһ¬мӮ¬мҡ© мқҙлһҖ,
    мқҙм „м—җ м •мқҳн•ң variableкіј function, class л“ұмқ„ лӢӨмӢң codeлЎң м •мқҳн•ҳм§Җ м•Ҡкі  лӢӨмӢң мӮ¬мҡ©н•ҳлҠ” кІғ мқ„ мқҳлҜён•Ё.
    moduleмқҙ м—ҶлӢӨл©ҙ мқҙм „м—җ мһ‘м„ұн•ң codeл“Өмқ„ л§ӨлІҲ copy and paste мӢңнӮӨкұ°лӮҳ кё°м–өн•ҳкі  лӢӨмӢң typingмқ„ н•ҙм•јн•  кІғмқҙлӢӨ.

    codeмқҳ мһ¬мӮ¬мҡ©мқ„ мң„н•ҙ PythonмқҖ мқҙм „м—җ кө¬нҳ„н–Ҳкұ°лӮҳ лӢӨлҘё мқҙк°Җ кө¬нҳ„н•ң .pyнҢҢмқјмқҳ мқҙлҰ„мңјлЎң
    н•ҙлӢ№ module (.pyлҘј к°ҖлҘҙнӮҙ)мқ„
    import statementлҘј мӮ¬мҡ©н•ҳм—¬ importн•ңлӢӨ.

    • ds_cal.py лқјлҠ” fileм—җ мһҲлҠ” codeлҘј мӮ¬мҡ©н•ҳкі  мӢ¶мқҖ кІҪмҡ°, ds_cal moduleмқ„ importн•ңлӢӨ.
    • import ds_cal кіј к°ҷмқҖ import statementлҘј мқҙмҡ©н•Ё.
    • importн•ҳл©ҙ ds_calмқҙлқјлҠ” variableмқҙ л§Ңл“Өм–ҙм§Җл©° мқҙлҠ” module class (or type)мқҳ objectлҘј м°ёмЎ°н•ҳкІҢ лҗңлӢӨ.

    Pythonм—җм„ңлҠ” мқҙлҜё standard libraryлЎң

    • built-in module мқ„ м ңкіөн•ҳкі  мһҲмқҢ.
    • (Pythonмқҳ built-in functionл“ӨмқҖ __builtins__ лқјлҠ” лӘЁл“Ҳм—җ мҶҚн•Ё)

    мқҙ мҷём—җлҸ„

    • Third partyм—җм„ң л§Ңл“  libraryмқҳ moduleл“Өкіј
    • к°ңл°ңмһҗк°Җ л§Ңл“  ліёмқёмқҳ moduleл“ӨлҸ„ import к°ҖлҠҘн•ҳлӢӨ.

    Note: Module Search Path

    • moduleмқ„ importн•  л•Ң, н•ҙлӢ№ moduleмқ„ мқҪм–ҙмҷҖм•јн•ҳлҠ”лҚ° мқҙ кІҪмҡ° Module Search Path лҘј м°ёкі н•ҳм—¬ pythonмқҖ moduleмқ„ мқҪм–ҙл“Өмһ„.
    • Module Search Path м—җ кё°мһ¬лҗң pathл“Өм—җм„ң moduleмқҳ мқҙлҰ„мқ„ кё°мӨҖмңјлЎң moduleмқ„ мқҪм–ҙл“Өмһ„ (к°ҖмһҘ лЁјм Җ мң„м№ҳн•ң pathмқҳ moduleм—җ мҡ°м„ к¶Ңмқҙ мһҲмқҢ).
    • кё°ліём ҒмңјлЎң нҳ„мһ¬ мһ‘м—… л””л үнҶ лҰ¬к°Җ нҸ¬н•ЁлҗҳлҜҖлЎң мқҙлҘј мқҙмҡ©н•ҳм—¬ moduleм—җ лҢҖн•ҙ мқөнһҲкі , лӢӨлҘё pathмқҳ moduleл“Өмқ„ мӮ¬мҡ©н•ҳкё° мң„н•ҙ Module Search Path лҘј мҲҳм •н•ҳлҠ” л°©лІ•мқҖ лӢӨмқҢ URLмқ„ мӮҙнҺҙліёлӢӨ: Module Search Path and sys.path.

    м°ёкі лЎң

    • нқ”нһҲ, importк°Җ лҗҳлҠ” .py нҢҢмқјмқ„ moduleмқҙлқјкі  л¶ҖлҘҙкі 
    • python interpreterм—җкІҢ мҲҳн–үлҗҳкё° мң„н•ҙ мқёмһҗлЎң л„ҳкІЁм§ҖлҠ” .py нҢҢмқјмқҖ main program нҳ№мқҖ main script лқјкі  л¶ҲлҰ°лӢӨ.

    м°ёкі лЎң __name__ мқҖ .pyнҢҢмқј мҰү module лӮҙм—җм„ң Python interpreterк°Җ н•ҙлӢ№ moduleм—җ н• лӢ№н•ң мқҙлҰ„, мҰү мһҗкё°мһҗмӢ мқҳ module nameмқ„ к°’мңјлЎң к°Җм§ҖлҠ” global variableмһ„.
    н•ҙлӢ№ main scriptлЎң лҸҷмһ‘н•ҳлҠ” кІҪмҡ°м—” __main__мқҙлқјлҠ” л¬ёмһҗм—ҙмқ„ к°’мңјлЎң к°Җм§җ.

    лӢӨмқҢмқҳ мҳҲлҠ” ds_cal.pyлҘј moduleлЎң importн•ҳлҠ” ds_run.pyлҘј нҶөн•ҙ к°„лӢЁн•ң module мӮ¬мҡ©лІ•мқ„ ліҙм—¬мӨҖлӢӨ.

    • ds_cal.pyмҷҖ ds_run.pyлҠ” лӘЁл‘җ к°ҷмқҖ directoryм—җ мң„м№ҳн•Ё.
    • мқҙ л‘җ нҢҢмқјмқҙ мһҲлҠ” directoryк°Җ current working directoryмһ„.

    main()

    • мқҙлҘј нҶөн•ҙ main scriptнҢҢмқјм—җм„ңлҠ” ds_cal.py нҢҢмқјмқҳ global scopeм—җ мһҲлҠ” attributeл“Өм—җ м ‘к·јмқ„ н•  мҲҳ мһҲмқҢ.
    • мӢӨм ңлЎң ds_cal.ds_addition(a, b)лҘј нҳём¶ңн•ҳм—¬ к·ё кІ°кіјлҘј rм—җ н• лӢ№н•Ё (лӘЁл“Ҳ ds_calмқҖ кі мң мқҳ namespaceлҘј к°Җм§Җл©° н•ҙлӢ№ namespaceм—җ м ‘к·јн•ҳкё° мң„н•ҙм„ң ds_calмқҙлқјлҠ” лӘЁл“ҲлӘ…мқ„ prefixлЎң к°Җм§җ).
    • moduleмқҳ global attributeл“Өм—җ м ‘к·јн•ҳкё°мң„н•ҙм„ңлҠ” module мқҙлҰ„мқ„ prefixлЎң мӮ¬мҡ©н•ҙм•ј н•Ё(мң„мқҳ мҳҲм—җм„ңлҠ” ds_calмқҙ prefixлЎң мӮ¬мҡ©лҗЁ).
    • мқҙлҠ” ds_calкіј к°ҷмқҖ importлҗң moduleм—җ лҢҖн•ң Name Space к°Җ мғқм„ұлҗҳм—ҲмқҢмқ„ мқҳлҜён•ҳл©°, н•ҙлӢ№ name spaceм—җ лӘЁл“Ҳмқҳ attributeл“Өмқҙ мҶҚн•ҳкІҢ лҗЁ.

    . operator¶

    ds_cal.ds_addition(a, b)м—җм„ң . (dot)мқҖ мқјмў…мқҳ operator мһ„.

    • .мқҳ мҷјмӘҪм—җ мһҲлҠ” variableмқҙ к°ҖлҘҙнӮӨлҠ” objectлҘј м°ҫкі ,
    • н•ҙлӢ№ objectм—җм„ң .мқҳ мҳӨлҘёмӘҪм—җ мһҲлҠ” nameм—җ н•ҙлӢ№н•ҳлҠ” object(=attribute)лҘј м°ҫм•„лӮҙм–ҙ м ‘к·јн•Ё.

    мң„мқҳ кІҪмҡ°, ds_calмқҙлқјлҠ” variableм—җ н• лӢ№лҗң objectм—җм„ң ds_additionмқҙлқјлҠ” function attributeлҘј callн•ҳкІҢ лҗЁ.

    importлҗҳлҠ” кІҪмҡ° moduleмқҖ execution(мҲҳн–ү)мқҙ мқҙлЈЁм–ҙм§җ.¶

    moduleмқҳ statementл“ӨмқҖ
    importк°Җ лҗҳлҠ” кІҪмҡ°м—җлҸ„ м—ӯмӢң мҲҳн–үмқҙ лҗҳлҜҖлЎң
    мқҙлҘј нҶөн•ҙ н•ҙлӢ№ moduleмқҳ мҙҲкё°нҷ”к°Җ к°ҖлҠҘн•Ё.

    • к°ҷмқҖ moduleмқҳ importк°Җ м—¬лҹ¬ лІҲ лҗҳм–ҙлҸ„, н•ңлІҲл§Ң мҲҳн–ү мқҙ лҗңлӢӨ (мөңмҙҲлЎң import лҗҳлҠ” кІҪмҡ°).
    • мҲҳн–үмқҙ лҗҳл©ҙм„ң н•ҙлӢ№ moduleмқҙ мҙҲкё°нҷ”к°Җ мқҙлЈЁм–ҙм§җ.

    main scriptм—җм„ңл§Ң мҲҳн–үлҗҳлҠ” л¶Җл¶„мқҙ лҗңлӢӨ.

    __name__ variableмқҖ н•ҙлӢ№ moduleмқҳ nameм—җ н•ҙлӢ№н•ҳлҠ” л¬ёмһҗм—ҙмқ„ к°’мңјлЎң к°Җм§ҖлҠ” нҠ№лі„н•ң variableмһ„.

    лӢӨмӢң н•ңлІҲ к°•мЎ°н•ҳм§Җл§Ң,

    • import л¬ёмқҙ лҸҷмқјн•ң moduleм—җ лҢҖн•ҙ нҳём¶ңмқҙ м—¬лҹ¬лІҲ лҗҳм–ҙлҸ„ мөңмҙҲ н•ңлІҲл§Ң мҲҳн–ү мқҙ лҗңлӢӨ.
    • л•Ңл¬ём—җ нҠ№м • Python Interpreter Sessionмқҙ мҲҳн–үлҗҳкі  мһҲлҠ” лҸҷм•Ҳм—җ нҠ№м • moduleмқҳ мҶҢмҠӨмҪ”л“ң мһҗмІҙк°Җ л°”лҖҢлҚ”лқјлҸ„ н•ҙлӢ№ ліҖкІҪмӮ¬н•ӯмқҙ л°ҳмҳҒлҗҳм§Җ м•ҠлҠ”лӢӨ.

    мқҙлҘј мң„н•ҙм„ңлҠ” importlib лӘЁл“Ҳмқҳ reload(лӘЁл“ҲлӘ…) functionмқ„ мқҙмҡ©н•ҙм•ј н•ңлӢӨ.
    мһҗм„ён•ң мӮ¬мҡ©лІ•мқҖ лӢӨмқҢ URLмқ„ м°ёкі н•ҳлқј.

    • лӘЁл“Ҳ ліҖкІҪмӮ¬н•ӯ лҸҷм ҒмңјлЎң л°ҳмҳҒн•ҳкё°: importlib.reload

    мЈјмқҳ: reassignment.¶

    PythonмқҖ standard libraryмқҳ moduleмқҳ attributesм—җ мһ¬н• лӢ№мқҙ (л§Өмҡ° мүҪкІҢ) к°ҖлҠҘн•ҳлӢӨлҠ” лӢЁм җмқ„ к°Җм§җ.

    лӢӨмқҢ codeлҠ” math moduleмқҳ piлқјлҠ” variableм—җ
    reassignmentк°Җ к°ҖлҠҘн•Ёмқ„ ліҙмһ„.

    import math

print(math.pi)

math.pi = 72

print(math.pi)

    piлҠ” \(\pi\) мқҳ к°’мқ„ к°Җм ём•јн•ҳлҠ”лҚ°
    мқҙк°ҷмқҖ reassignmentлҠ” л§ҺмқҖ л¬ём ңмқҳ мӣҗмқёмқҙ лҗ  мҲҳ мһҲлӢӨ.

    мӢӨм ң moduleм—җ н•ҙлӢ№н•ҳлҠ” fileм—җм„ң мҲҳм •мқҙ мқјм–ҙлӮҳм§„ м•Ҡкі , лӢЁмҲңнһҲ memoryмғҒм—җм„ң mathмқҳ namespaceмқҳ piлқјлҠ” мқҙлҰ„мқҙ к°ҖлҘҙнӮӨлҠ” objectк°Җ л°”лҖҗ кІғмқҙлҜҖлЎң н•ҙлӢ№ Python interpreterмқҳ sessionм—җм„ңл§Ң 72лқјлҠ” к°’мқ„ мқҳлҜён•ҳкІҢ лҗҳкі , лӢӨлҘё math лӘЁл“Ҳмқ„ мӮ¬мҡ©н•ҳлҠ” sessionм—җм„ңлҠ” л¬ём ңк°Җ м—ҶлӢӨ.

    Pythonм—җм„ңлҠ” мқҙмІҳлҹј кі м •лҗҳм–ҙм•ј н•ҳлҠ” мғҒмҲҳлҘј freezeн•  мҲҳк°Җ м—ҶлӢӨлҠ” лӢЁм җмқ„ к°Җм§Җкі  мһҲмқҢ.

    asлҘј нҶөн•ң alias(лі„м№ӯ) мӮ¬мҡ©н•ҳкё°.¶

    м•һм„ң мҳҲмқҳ import statementм—җм„ң importлҗң лӘЁл“Ҳм—җ м ‘к·јмқҖ ***лӘЁл“Ҳ нҢҢмқјмқҳ name(мқҙлҰ„)***мқ„ нҶөн•ҙм„ң мқҙлЈЁм–ҙм§җ.

    н•ҙлӢ№ мқҙлҰ„мқҙ л„Ҳл¬ҙ кёҙ кІҪмҡ° лӮҳ кҙҖлЎҖм ҒмңјлЎң мӮ¬мҡ©лҗҳлҠ” лі„м№ӯ мқҙ мһҲлҠ” кІҪмҡ°м—”
    aliasлҘј м§Җм • н•ҙмЈјм–ҙ лӘЁл“Ҳмқҳ attributeм—җ м ‘к·јн•Ё.

    н•ҙлӢ№ aliasлҘј м§Җм •н•ҳлҠ” import statementлҠ” лӢӨмқҢкіј к°ҷмқҢ.

    import ds_cal as dc

    Python м—җм„ң к°ҖмһҘ мң лӘ…н•ң alias мӨ‘ н•ҳлӮҳлҠ” NumPy лҘј к°ҖлҘҙнӮӨлҠ” npмһ„.
    NumPyлҘј мӮ¬мҡ©н•ҳлҠ” кІҪмҡ°, import numpy as np к°Җ к°ҖмһҘ л§Һмқҙ мӮ¬мҡ©лҗЁ.

    мӣҗн•ҳлҠ” attributeл§Ң importн•ҳкё°.¶

    нҳ„мһ¬ ds_cal.py лӘЁл“Ҳм—җм„ң н•„мҡ”н•ң attributeлҠ” л°”лЎң ds_addition н•ЁмҲҳ лҝҗмқҙлқјкі  к°Җм •н•ҳмһҗ.

    Packages¶

    Pythonм—җм„ң moduleмқҙ .pyнҢҢмқјлЎң кө¬нҳ„лҗҳлҠ” кІғмІҳлҹј,
    packageлҠ” moduleл“Өмқҙ нҸ¬н•Ёлҗң directoryлЎң кө¬нҳ„лҗңлӢӨ.

    Python 2.x мқҳ кІҪмҡ°, __init__.pyк°Җ мһҲм–ҙм•ј н–ҲмңјлӮҳ
    Python 3.xм—җм„ңлҠ” н•„мҡ”к°Җ м—ҶмқҢ.

    н•ҳм§Җл§Ң нҳёнҷҳм„ұмқ„ мң„н•ҙ л№Ҳ нҢҢмқјлЎңлқјлҸ„ __init__.pyлҘј л„Јм–ҙмЈјлҠ” кІҢ мўӢмқҢ.

    packageлҘј нҶөн•ҙ moduleл“ӨмқҖ

    • logically л№„мҠ·н•ң кІғл“ӨлҒјлҰ¬ кө¬л¶„м§“кі 
    • кі„мёөнҷ” н•  мҲҳ мһҲлӢӨ.

    лӢӨмқҢ мҳҲм ңлҠ”
    мӮ¬м№ҷм—°мӮ°мқ„ кө¬нҳ„н•ң ds_cal.pyлҘј farith packageм—җ л‘җкі ,
    кё°ліё мӮјк°Ғн•ЁмҲҳ кҙҖл Ё н•ЁмҲҳлҘј м ңкіөн•ҳлҠ” ds_cal.pyлҘј ftri packageлЎң лӮҳлҲ„м–ҙ м ңкіөн•ҳл©ҙм„ң
    мқҙл“Өмқ„ ds_run03.pyм—җм„ң лӘЁл“ҲлЎң мӮ¬мҡ©н•ҳлҠ” кІғмқ„ ліҙм—¬мӨҢ.

    Namespace Package¶

    м—¬лҹ¬ лҸ…лҰҪм Ғмқё нҢЁнӮӨм§Җл“Өмқ„ н•ҳлӮҳмқҳ "л…јлҰ¬м Ғмқё нҢЁнӮӨм§Җ" (лӢЁмҲңнһҲ н•ҳлӮҳмқҳ namespace м—җ мҶҚн•ҳлҸ„лЎқ н•ҙмЈјлҠ” м—ӯн• мқҳ нҢЁнӮӨм§Җ) мІҳлҹј ліҙмқҙкІҢ н•ҙмЈјкё° мң„н•ҙ м ңм•Ҳлҗң лҸ„кө¬.

    • лҢҖк·ңлӘЁ н”„лЎңм қнҠём—җм„ң лҸ…лҰҪм Ғмқё нҢЁнӮӨм§Җ кҙҖлҰ¬к°Җ к°ҖлҠҘн•Ё.

    namespace packageмқҳ кІҪмҡ°,

    • л¬јлҰ¬м ҒмңјлЎң лӢӨлҘё кІҪлЎң(к°Ғк°Ғмқҳ sys.pathм—җ itemмқҙм–ҙм•јн•Ё.)м—җ мһҲлҚ”лқјлҸ„,
    • к°ҷмқҖ нҢЁнӮӨм§ҖлӘ…мқ„ кіөмң н•  кІҪмҡ° н•ҳлӮҳмқҳ packageлЎң Pythonм—җм„ң мІҳлҰ¬лҗЁ.
    • м—¬лҹ¬ лӢӨлҘё к°ңл°ңк·ёлЈ№мқҙ лҸҷмқјн•ң нҢЁнӮӨм§ҖлҘј лҸҷмӢңм—җ нҷ•мһҘн•ҳм—¬ к°ңл°ңн•ҳлҠ” кІғмқ„ к°ҖлҠҘн•ҳкІҢ н•Ё.

    мқҙ кІҪмҡ°, к°ҷмқҖ нҢЁнӮӨм§ҖлӘ…мқ„ кіөмң н•ҳлҠ” лӘЁл“  subdirectory лӮҙм—җ __init__.pyк°Җ мһҲм–ҙм„  м•ҲлҗЁ (Python 3.3 мқҙмғҒл¶Җн„°л§Ң мӮ¬мҡ©к°ҖлҠҘн•Ё).

    sys.pathм—җ м •мқҳлҗң кІҪлЎңл“Өмқ„ нғҗмғүн•ҳл©ҙм„ң namespace packageлЎң мқёмӢқлҗң лӘЁл“  л””л үнҶ лҰ¬л“Өмқ„ мІҳлҰ¬н•  л•Ң, лҸҷмқјн•ң мқҙлҰ„мқҳ л””л үнҶ лҰ¬лҠ” к°ҷмқҖ "л…јлҰ¬м Ғ нҢЁнӮӨм§Җ" лЎң лі‘н•©н•Ё.

    лӢӨмқҢмқҳ л§ҒнҒ¬м—җм„ң мҳҲлҘј л“Өм–ҙ м„ӨлӘ…н•ҳкі  мһҲмңјлӢҲ м°ёкі н•  кІғ:
