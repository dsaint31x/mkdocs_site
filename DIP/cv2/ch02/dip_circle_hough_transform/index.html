    Circle Hough Transform¶

    Hough TransformмқҖ imageм—җм„ң line, circle л“ұмқҳ к°„лӢЁн•ң лҸ„нҳ•мқ„ м°ҫлҠ”лҚ° мӮ¬мҡ©лҗҳлҠ” л°©лІ•мһ„.

    Required Parameters : Circle Equation¶

    мҡ°м„  circleмқ„ м§Җм •н•ҳлҠ” parameterл“Өмқҙ л¬ҙм—Үмқём§ҖлҘј мӮҙнҺҙліҙкё° мң„н•ҙ мӣҗмқҳ н‘ңмӨҖ л°©м •мӢқмқ„ мӮҙнҺҙліёлӢӨ.

    \[ (x-x_c)^2 + (y-y_c)^2 = r^2 \]

    where

    • \(x,y\) : imageм—җм„ңмқҳ x-axis, y-axisм—җ лҢҖн•ң coordinate.
    • \(x_c, y_c\) : circleмқҳ centerм—җм„ңмқҳ x-axis, y-axis coordinate.
    • \(r\) : radius.

    мҰү, \(x_c, y_c, r\) лЎң мқҙлЈЁм–ҙм§„ 3к°ңмқҳ parameterк°Җ мһҲмңјл©ҙ нҠ№м • circleмқ„ м§Җм •н•  мҲҳ мһҲмқҢ.

    Hough Space¶

    мқҙм „м—җ лӢӨлЈ¬ Line Hough Transformм—җм„ң radial distanceмҷҖ angleмқҙ lineмқ„ м§Җм •н•ҳлҠ” parameterл“Өмқҙм—Ҳкё° л•Ңл¬ём—җ 2-Dimensional Parameter Spaceк°Җ Hough transformмқҳ кІ°кіјмҳҖлҚҳ кІғмІҳлҹј, circleмқҳ кІҪмҡ° parameterк°Җ 3к°ңк°Җ лӮҳмҳӨкё° л•Ңл¬ём—җ 3-Dimensional Parameter Space (or Hough space)к°Җ ліҖнҷҳмқҳ кІ°кіјк°Җ лҗЁ.

    Preprocessing¶

    мқјл°ҳм ҒмңјлЎң Hough Transformмқ„ мӢӨм ңлЎң мҲҳн–үн•ҳкё° м „м—җ мқҙлЈЁм–ҙм§ҖлҠ” м „мІҳлҰ¬лҠ” circleмқ„ м°ҫлҠ” кІҪмҡ°лӮҳ lineмқ„ м°ҫлҠ” кІҪмҡ°лӮҳ нҒ° м°Ёмқҙк°Җ м—ҶмқҢ.

    Edge Detection (or extraction)
    Canny algorithmмқ„ нҶөн•ҙ edgeлҘј м¶”м¶ңн•ҳм—¬ edgeлҠ” 1, лӮҳлЁём§„ 0 мқё binary imageлҘј м–»м–ҙлғ„.

    The case of fixed radius¶

    мҡ°м„  radiusлҘј кі м •н•ҳм—¬ мӣҗмқҳ мӨ‘м җ мўҢн‘ңл§Ңмқ„ кө¬н•ҳлҠ” кІҪмҡ°м—җм„ң м¶ңл°ңн•ңлӢӨ. (ліҙлӢӨ лӢЁмҲңн•ң кІҪмҡ°м—җм„ң м¶ңл°ңн•ҳлҠ” кІҢ мқҙн•ҙм—җ лҸ„мӣҖмқҙ лҗҳлҜҖлЎң...)

    Canny algorithmмңјлЎң м–»м–ҙлӮё edgeк°Җ 1мқҙкі  лӮҳлЁём§„ 0мқё binary imageм—җм„ң edgeмқё pixelмқ„ мӨ‘м җмңјлЎң мӮјм•„ кі м •лҗң radiusлЎң circleмқ„ кІ°м •н•ҳкі  н•ҙлӢ№ circleм—җ мҶҚн•ҳлҠ” pixelл“Өмқҳ к°’мқ„ 1м”© мҰқк°ҖмӢңнӮЁлӢӨ.

    • edgeк°Җ м¶”м¶ңлҗң binary imageмҷҖ к°ҷмқҖ widthмҷҖ heightлҘј к°Җм§Җл©ҙм„ң лӘЁл“  pixelмқҳ к°’мқҙ 0мқё imageлҘј мғқм„ұ.
    • н•ҙлӢ№ imageм—җ лҢҖн•ҙ edgeмң„м№ҳлҘј мӨ‘м җмңјлЎң н•ҳлҠ” circleмқ„ к·ёлҰј (circleм—җ мҶҚн•ҳлҠ” pixelк°’л“Өмқ„ 1м”© мҰқк°ҖмӢңнӮҙ)

    edgeм—җ мҶҚн•ҳлҠ” pixelмқҙ a,bм—җ лҢҖн•ҙ 2к°ңмқҳ кі м •лҗң radiusмқҳ circleмқҙ 2к°ң к·ёл Өм§ҖкІҢ лҗҳкі , 2к°ңмқҳ circleм—җм„ң лҸҷмӢңм—җ мҶҚн•ҳлҠ” pixelмқҖ к°’мқҙ 2к°Җ лҗ  кІғмһ„.

    лӢӨмқҢ к·ёлҰјмқҖ мқҙ лҸҷмһ‘мқ„ ліҙм—¬мӨҢ.

    • мҷјмӘҪмқҳ 3к°ңмқҳ м җмқҖ н•ҳлӮҳмқҳ мӣҗм—җ мҶҚн•ҳлҠ” м җл“ӨлЎң canny edge detectionмқ„ нҶөн•ҙ мқҙл“Ө 3к°ңмқҳ м җл§Ң 1мқҙкі  лӮҳлЁём§„ 0мқё binary imageк°Җ м–»м–ҙмЎҢлӢӨкі  к°Җм •н•Ё.
    • 3к°ңмқҳ м җл“ӨмқҖ к°ҷмқҖ мӨ‘мӢ¬мқ„ кіөмң н•ҳлҜҖлЎң мқҙл“Өмқ„ к°Ғк°Ғ мӨ‘м җмңјлЎң мӣҗмқ„ к·ёлҰ¬л©ҙ мӨ‘мӢ¬м—җ н•ҙлӢ№н•ҳлҠ” pixelмқҳ к°’мқҖ 3мқҙ лҗЁ.
    • л§Ңмқј canny edge detectionмңјлЎң мқҙлҜём§Җ кіөк°„м—җм„ң мӣҗлһҳмқҳ мӣҗм—җ н•ҙлӢ№н•ҳлҠ” лӘЁл“  pixelл“Өм—җ лҢҖн•ҙ Hough spaceм—җм„ң мӣҗмқ„ к·ёлҰ°лӢӨл©ҙ мӨ‘м җм—җ н•ҙлӢ№н•ҳлҠ” pixelмқҳ к°’мқҙ к°ҖмһҘ нҒ° к°’мқ„ к°Җм§ҖкІҢ лҗЁ (image spaceм—җм„ң circleм—җ мҶҚн•ҳлҠ” pixelмқҳ мҲ«мһҗмҷҖ к°ҷмқҖ к°’мқ„ к°Җм§җ)

    Hough transformмқҳ кІ°кіјлҘј нҠ№м • thresholdлЎң binary imageлЎң л§Ңл“Өл©ҙ, мӣҗмқҳ мӨ‘мӢ¬м—җ н•ҙлӢ№н•ҳлҠ” pixelл§Ң к°’мқ„ к°Җм§ҖлҠ” кІ°кіјлҘј м–»м–ҙлғ„.

    мқҙлҠ” radiusлҘј кі м •н•ҙлҶ“мқҖ кІҪмҡ°мқҙлҜҖлЎң мӨ‘мӢ¬мқҳ мўҢн‘ңлҘј м–»м—Ҳкё° л•Ңл¬ём—җ н•ҙлӢ№ мӣҗмқ„ м§Җм •н•ҳлҠ”лҚ° н•„мҡ”н•ң 3к°Җм§Җ нҢҢлқјл©”н„°лҘј лӘЁл‘җ кө¬н•ң м…Ҳмһ„.

    General case¶

    м•һм„ң м• кё°н•ң л¶Җл¶„мқҖ radiusк°Җ кі м •лҗң кІҪмҡ°лЎң, radiusлҘј кі м •н•ҳм§Җ м•ҠмқҖ кІҪмҡ°м—” 3м°Ёмӣҗмқҳ Hough spaceм—җм„ң accumulated valueлҘј к°Җм§ҖлҠ” voxelл“Өмқ„ м–»кІҢлҗңлӢӨ.

    мң„мқҳ к·ёлҰјмқҖ 3м°Ёмӣҗмқҳ Hough spaceлЎң radiusм—җ н•ҙлӢ№н•ҳлҠ” м¶•мқҙ м¶”к°Җлҗң кІғмқ„ ліҙм—¬мӨҖлӢӨ. rм¶•м—җ orthogonal planeл“Өмқ„ к°Ғк°Ғ мӮҙнҺҙліҙл©ҙ radiusк°Җ кі м •лҗҳм—ҲлҚҳ кІҪмҡ°м—җ н•ҙлӢ№н•ҳкІҢ лҗңлӢӨ.

    мқҙ кІ°кіјлҘј нҠ№м • thresholdлЎң binarizationмқ„ н•ҳл©ҙ мӣҗмқҳ мӨ‘мӢ¬м—җ н•ҙлӢ№н•ҳлҠ” voxelл“Өмқ„ м–»мқ„ мҲҳ мһҲкі , н•ҙлӢ№ voxelмқҳ мўҢн‘ңм—җ н•ҙлӢ№н•ҳлҠ” \(c_x, c_y, r\)мқ„ нҶөн•ҙ мӣҗлһҳ imageм—җм„ңмқҳ circleмқ„ кІҖм¶ңн•  мҲҳ мһҲлӢӨ.

    src image мҷҖмқҳ л№„көҗмқҳ н•„мҡ”м„ұ¶

    лӢӨмқҢ к·ёлҰјмқ„ ліҙл©ҙ м—¬лҹ¬ мӣҗмқҙ мЎҙмһ¬н•ҳлҠ” кІҪмҡ°м—җ нҢҢлһҖмғү л„ӨлӘЁлЎң н‘ңмӢңлҗң pixelм—җм„ңлҸ„ мӨ‘мІ©мқҙ л°ңмғқн•ҳм—¬ м—ҶлҚҳ circleмқҙ кІҖм¶ңлҗ  мҲҳ мһҲмқҢмқ„ м•Ң мҲҳ мһҲмқҢ.

    л•Ңл¬ём—җ circleмқ„ мғқм„ұн•ҳкі  лӮҳм„ң src мқҙлҜём§ҖмҷҖмқҳ л№„көҗлҘј н•ҙм•јн•Ё.

    Hough Gradient Method¶

    OpenCVм—җм„ңлҠ” м•һм„ң м„ӨлӘ…н•ң л°©лІ• лҢҖмӢ , GradientлҘј мқҙмҡ©н•ҳлҠ” Hough gradient methodлҘј мӮ¬мҡ©н•Ё.

    CannyлЎң м–»мқҖ binary edge imageм—җм„ң edge pixelм—җм„ңмқҳ gradientл“Өмқ„ кі„мӮ°(Sobel filterмӮ¬мҡ©)н•ҳкі  н•ҙлӢ№ gradientм—җ н•ҙлӢ№н•ҳлҠ” м§Ғм„  (radial distanceмҷҖ thetaлЎң м§Җм •)мқ„ м§ҖлӮҳлҠ” pixelл“Өмқҳ к°’мқ„ 1лЎң мҰқк°ҖмӢңнӮӨлҠ” л°©мӢқмңјлЎң accumulationмқ„ мҲҳн–үн•ңлӢӨ. мқҙлҘј нҶөн•ҙ мӣҗмқҳ мӨ‘мӢ¬ нӣ„ліҙкө°мқ„ м–»мқ„ мҲҳ мһҲмқҢ. (нҠ№м • thresholdлЎң thresholdingмқ„ нҶөн•ҙ)

    м–»м–ҙлӮё мӣҗмқҳ мӨ‘мӢ¬ нӣ„ліҙкө°мқҳ лӘЁл“  мң„м№ҳл“Ө к°Ғк°Ғм—җ лҢҖн•ҙ, edge binary imageмқҳ pixelк°„мқҳ кұ°лҰ¬лҘј кө¬н•ҳкі  мқҙлҘј radiusлЎң мӮјм•„ н•ҙлӢ№н•ҳлҠ” \(x_c, y_c, r\)мқҳ voxelмқҳ к°’мқ„ мҰқк°ҖмӢңнӮӨлҠ” кіјм •мқ„ мҲҳн–үн•ңлӢӨ.

    мң„мқҳ кіјм •мқ„ л§Ҳм№ҳкі лӮҳм„ң нҠ№м • thresholdлЎң binarizationмқ„ мҲҳн–үн•ҳл©ҙ circleм—җ н•ҙлӢ№н•ҳлҠ” \(x_c, y_c, r\)мқ„ кө¬н•  мҲҳ мһҲмқҢ.

    м•һм„ң мӮҙнҺҙліё кІҪмҡ°мҷҖ л§Ҳм°¬к°Җм§ҖлЎң src мқҙлҜём§ҖмҷҖ л№„көҗлҘј н•ҙм„ң к°Җм§ңмқё кІҪмҡ°лҠ” м ңкұ°н•ҙм•јн•Ё.

    Code Example for OpenCV¶

    OpenCVлҠ” cv2.HoughCirclesлҘј нҶөн•ҙ CHTлҘј м ңкіөн•ңлӢӨ.
    (лӮҙл¶Җм ҒмңјлЎң Canny edge detectionмқ„ мҲҳн–үн•Ё.)

    circles = cv2.HoughCircles(
    image, 
    method, 
    dp, 
    minDist
    [, param1
    [, param2
    [, minRadius
    [, maxRadius
    ]]]]])
    • image: 8-bit gray-scale input image
    • method: HOUGH_GRADIENT л§Ң м„ нғқк°ҖлҠҘн•Ё. --;;
    • dp : The inverse ratio of accumulator resolution and image resolution. 2мқј кІҪмҡ°, Hough spaceмқҳ widthмҷҖ heightк°Җ мӣҗліёлҢҖл№„ ½лЎң мӨ„м–ҙл“ лӢӨ. к°’мқҙ нҒҙмҲҳлЎқ scaleмқҙ м»Өм ём„ң нҒ° нҢЁн„ҙмқ„ м°ҫкІҢ лҗЁ.
    • minDist: Minimum distance between the centers of the detected circles. к°’мқҙ мһ‘мқ„мҲҳлЎқ кІҖм¶ңлҗҳлҠ” circleмқҳ мҲҳк°Җ мҰқк°Җн•ҳлӮҳ кұ°м§“ circleмқ„ м°ҫмқ„ нҷ•лҘ лҸ„ м»Өм§җ. 0мқ„ м„ нғқн•  мҲҳ м—Ҷмңјл©° мқҙлҠ” OpenCVмқҳ HoughCirclesк°Җ лҸҷмӢ¬мӣҗмқ„ кІҖм¶ңн•ҳм§Җ лӘ»н•ҳлҠ” лӢЁм җмңјлЎң мқҙм–ҙм§җ.
    • param1: The higher threshold of the two passed to the Canny edge detector (the lower canny threshold is twice smaller). к°’мқҙ мһ‘мқ„мҲҳлЎқ кІҖм¶ңлҗҳлҠ” circleмқҳ мҲҳк°Җ мҰқк°Җ.
    • param2: The accumulator threshold for the circle centers at the detection stage. к°’мқҙ мһ‘мқ„мҲҳлЎқ кІҖм¶ңлҗҳлҠ” circleмқҳ мҲҳк°Җ мҰқк°Җн•ҳлӮҳ кұ°м§“ circleмқ„ м°ҫмқ„ нҷ•лҘ лҸ„ мҰқк°Җн•Ё.
    • minRadius: minimum radius that you expect. If unknown, put zero as default. maxRadius: Default is 0. If <= 0, uses the maximum image dimension. If < 0, returns centers without finding the radius.

    л°ҳнҷҳлҗҳлҠ” circlesлҠ” \(1 \times N \times 3\)мқҳ float listлЎң NмқҖ кІҖм¶ңлҗң circleмқҳ мҲҳмқҙкі , к°Ғк°Ғм—җ лҢҖн•ҙ \(c_x, c_y, r\)лЎң кө¬м„ұлҗңлӢӨ.

    лӢӨмқҢмқҖ Test exampleмһ„.

    import numpy as np
import matplotlib.pyplot as plt
import cv2

# test image к·ёлҰ¬кё°
img = np.zeros((512,512,3),dtype=np.uint8)
tmp = img.copy()
tmp = cv2.circle(tmp, (200,200), 50, [255,0,0], -1)
tmp = cv2.circle(tmp, (200,200), 20, [0,0,255], -1)
tmp = cv2.circle(tmp, (400,400), 50, [255,0,0], -1)
img = tmp.copy()
plt.imshow(tmp[:,:,::-1])

# grayscale imageкө¬н•ҳкё°.
tmp = img.copy()
src = cv2.cvtColor(tmp, cv2.COLOR_BGR2GRAY)
# edges = cv2.Canny(tmp, 100, 200)
# blur = cv2.GaussianBlur(edges, (5,5),0)
blur = cv2.GaussianBlur(src, (5,5),0)

plt.imshow(blur,cmap='gray')

# CHT мҲҳн–ү 
circles = cv2.HoughCircles(blur, 
                           cv2.HOUGH_GRADIENT,
                           1,
                           minRadius=20,
                           maxRadius=90,
                           minDist=50,
                           circles=None,
                           param1=100,
                           param2=10 
                           )
print(np.array(circles).shape) # num_of_circles x 1 x (c_x,c_y,r)

# кІ°кіј нҷ•мқё
# detected circleм—җ мҙҲлЎқмғүмңјлЎң н‘ңмӢңн•ҳкі , мӨ‘мӢ¬мқҖ л¶үмқҖ мғүмңјлЎң н‘ңмӢң.
tmp = img.copy()
if circles is not None:
  circles = np.array(np.around(circles), dtype=int)
  print(circles.shape)
  for c in circles[0]:
    print(c)
    cv2.circle(tmp, (c[0],c[1]), c[2], (0,255,0), 2)
    cv2.circle(tmp, (c[0],c[1]), 2, (0,255,0), 2)
plt.imshow(tmp)
