    Contour Features¶

    Moment¶

    Image momentsмқҖ л¬јмІҙмқҳ мӨ‘мӢ¬, л¬јмІҙмқҳ л©ҙм Ғ л“ұмқ„ кі„мӮ°н•ҳлҠ”лҚ° мқҙмҡ©лҗҳлҠ” quantity(м–‘)мһ„.

    pixel intensity(вҶҗл¬јлҰ¬м—җм„ң force, massл“ұ)мқҳ м •лҹүм Ғ нҒ¬кё° мҷҖ н•Ёк»ҳ л¶„нҸ¬ (м–ҙл–Ө кё°мӨҖм—җ лҢҖн•ң) лҘј кі л Өн•ң м •лҹүм Ғ м§Җн‘ң.
    мЈјлЎң gray-scale or binary imageм—җм„ң мӮ¬мҡ©лҗЁ.

    moment к°Җ к°Җм§ҖлҠ” лң»мқ„ мўҖ лҚ” мӮҙнҺҙліҙл Өл©ҙ лӢӨмқҢ URLмқ„ м°ёкі н•  кІғ : л¬јлҰ¬лҹүмқҳ кҙҖм җм—җм„ң momentлһҖ?

    Spatial Moment (or raw Moment)¶

    \[m_{pq} = \sum_y\sum_x x^p y^q I(x, y) \]
    • \(p,q\) : degree(м°ЁмҲҳ)м—җ н•ҙлӢ№н•Ё. 0 мқҙмғҒмқҳ м •мҲҳк°Җ л§Һмқҙ мӮ¬мҡ©лҗЁ.
    • \(x,y\) : pixelмқҳ x, y мўҢн‘ңк°’
    • \(I(x,y)\) : x,y мң„м№ҳмқҳ pixel intensity.

    (raw) momentмқҳ к°’мқҖ pixel intensity лҝҗ м•„лӢҲлқј pixelмқҳ (м ҲлҢҖ)мң„м№ҳм—җ л§Өмҡ° нҒ° мҳҒн–Ҙмқ„ л°ӣлҠ”лӢӨ.
    ліҙнҶө мӣҗм җмқ„ кё°мӨҖмңјлЎң кі„мӮ°лҗңлӢӨ.

    Central Moment¶

    м•һм„ң ліё spatial momentлҠ” pixelмқҳ мўҢн‘ңк°’м—җ мҳҒн–Ҙмқ„ нҒ¬кІҢ л°ӣкё° л•Ңл¬ём—җ, мқҙлҜём§Җ лӮҙм—җм„ңмқҳ м ҲлҢҖ мң„м№ҳм—җ л”°лқј к°’мқҙ л§Һмқҙ л°”лҖҢкІҢ лҗңлӢӨ.

    мқҙк°ҷмқҖ raw momentмҷҖ лӢ¬лҰ¬, pixel к°’мқҳ л¶„нҸ¬лЎң кІ°м •лҗҳлҠ” shapeм—җ dependentн•ҳл©ҙм„ң, м ҲлҢҖ мң„м№ҳм—җм„ң лҢҖн•ҙм„ңлҠ” к°Җкёүм Ғ лҸ…лҰҪм Ғмқё м •лҹүм Ғ м§Җн‘ңк°Җ мһҲлӢӨл©ҙ, нҠ№м • shapeмқҳ objectлҘј кІҖм¶ңм—җ мң лҰ¬н•ҳлӢӨ.

    мқҙк°ҷмқҖ нҠ№м„ұмңјлЎң м ңм•Ҳлҗң кІғмқҙ л°”лЎң Central MomentмқҙлӢӨ.

    \[\mu_{pq} = \sum_y\sum_x (x-\bar{x})^p (y-\bar{y})^q I(x, y) \]

    \(\bar{x},\bar{y}\) : x, yмқҳ meanмңјлЎң мӨ‘мӢ¬(imageмқҳ мӨ‘мӢ¬)м—җ н•ҙлӢ№н•ңлӢӨ.

    Normalized Central Moment¶

    мӨ‘мӢ¬ лӘЁл©ҳнҠёлҘј нҶөн•ҙ л‘җ мқҙлҜём§Җ мғҒмқҳ к°қмІҙк°Җ к°ҷмқҖм§Җ л№„көҗн•  мҲҳ мһҲмңјлӮҳ, мқҙлҜём§Җмқҳ л°°мңЁ л“ұм—җ л”°лқј central momentлҠ” к°ҷмқҖ objectм—җм„ңлҸ„ м°ЁмқҙлҘј к°Җм§Ҳ мҲҳ мһҲмқҢ. objectмқҳ м „мІҙ нҒ¬кё°лҘј лӮҳлҲ„м–ҙм„ң мўҖлҚ” robustн•ҳкІҢ л§Ңл“  кІғмқҙ л°”лЎң normalized central momentмһ„.

    \[\nu_{pq}= \frac{\mu_{pq}}{\mu_{00}^{\left(1+\frac{p+q}{2}\right)}}\]

    OpenCV м—җм„ң momentкө¬н•ҳкё°.¶

    OpenCVм—җм„ңлҠ” cv2.moment()м—җ objectмқҳ contourлҘј л„ҳкІЁмӨҢмңјлЎңм„ң 3м°Ёк№Ңм§Җмқҳ moment л°Ҹ кҙҖл Ё мҲҳм№ҳл“Өмқ„ кө¬н•  мҲҳ мһҲмқҢ.

    лӢӨмқҢ мҳҲм ңлҘј ліҙл©ҙ momentл“Өмқ„ кө¬н•ҳлҠ” л°©лІ•мқ„ нҷ•мқён•  мҲҳ мһҲлӢӨ.

    Hu Moment:Match Shapes¶

    objectмқҳ contourлҘј кё°л°ҳмңјлЎң objectк°„мқҳ shapeк°Җ мң мӮ¬н•ңм§Җ м—¬л¶ҖлҘј нҢҗм •н•  мҲҳ мһҲмқҢ.

    OpenCVлҠ” hu-momentлҘј кё°л°ҳмңјлЎң shapeмқҳ мң мӮ¬лҸ„лҘј мёЎм •н•ҳлҠ” cv2.matchShapesлҘј м ңкіөн•Ё.

    cnt1,
    cnt2,
    1,  # methods
    0.0 # 0.0мңјлЎң н•ӯмғҒ мһ…л Ҙ.
    )

    н•ҙлӢ№ л°©лІ•м—җ лҢҖн•ң кіөмӢқмқҖ лӢӨмқҢмқ„ м°ёкі  : ShapeMatchMode

    м„ёлІҲм§ё parameter methodлҠ” мң мӮ¬лҸ„ мёЎм •м—җ мӮ¬мҡ©н•  normмқ„ м§Җм •н•ңлӢӨ.

    • 0 : L1мқ„ кё°л°ҳ.
    • 1 : L2мқ„ кё°л°ҳ.
    • 2 : L3мқ„ кё°л°ҳ.

    мҰү, retмқҙ мһ‘мқ„мҲҳлЎқ л№„мҠ·н•ң shapeмһ„мқ„ мқҳлҜён•Ё.

    л§Ҳм§Җл§ү parameterлҠ” м„ёлІҲм§ё parameterм—җ м§Җм •н•ң methodм—җм„ң н•„мҡ”н•ң к°’мқ„ л„Јм–ҙмЈјкё° мң„н•ҙ н• лӢ№лҗҳм—Ҳм§Җл§Ң, м•„м§Ғ м ңлҢҖлЎң м§Җмӣҗлҗҳм§Җ м•ҠмңјлҜҖлЎң 0.0мқ„ л„Јм–ҙмӨҖлӢӨ.

    perimeter = cv2.arcLength(contour[0],True)

    • 2лІҲм§ё argumentлҠ” н•ҙлӢ№ contourк°Җ closedмқём§Җ м—¬л¶ҖлҘј лӮҳнғҖлғ„.

    Contour Approximation¶

    м •л°ҖлҸ„лҘј м§Җм •н•ҳм—¬, contourлҘј н•ҙлӢ№ м •л°ҖлҸ„ лӮҙм—җм„ң ліҙлӢӨ к°„лӢЁн•ҳкІҢ к·јмӮ¬н•  мҲҳ мһҲмқҢ.

    OpenCVм—җм„ңлҠ” мқҙлҘј мң„н•ҙ Douglas-Peucker Algorithmмқҳ кө¬нҳ„л¬јмқ„ м ңкіөн•Ё.

    лҢҖлһөм Ғмқё мӮ¬мҡ©лІ•мқҖ лӢӨмқҢкіј к°ҷмқҢ.

    # м „мІҙ л‘ҳл Ҳмқҳ 5%лЎң мҳӨм°Ё лІ”мң„ м§Җм • 
app_rate = 0.05

# н—Ҳмҡ©к°ҖлҠҘн•ң м •л°ҖлҸ„мқҳ м°Ёмқҙ = н—Ҳмҡ©к°ҖлҠҘн•ң contour arclength(л‘ҳл Ҳкёёмқҙ)
epsilon = app_rate * cv2.arcLength(contour, True) 

# к·јмӮ¬ contourкө¬н•ҳкё°.
approx = cv2.approxPolyDP(contour, 
                          epsilon, 
                          True)
    • contour : к·јмӮ¬м№ҳлҘј кө¬н•ҳкі мһҗн•ҳлҠ” objectмқҳ contour.
    • epsilon : contour arc lengthк°Җ м–ҙлҠҗм •лҸ„к№Ңм§Җ мӨ„м–ҙл“Ө мҲҳ мһҲлҠ”м§ҖлҘј лӮҳнғҖлӮҙлҠ” мҳӨм°ЁлІ”мң„.
    • True : contourк°Җ closed мқём§Җ м—¬л¶ҖлҘј м•Ңл ӨмЈјлҠ” argument

    л°ҳнҷҳлҗҳлҠ” approxк°Җ к·јмӮ¬мІҳлҰ¬лҗң contourмһ„.

    лӢӨмқҢ мҳҲм ңлҘј м°ёкі .

    import cv2
import numpy as np
import matplotlib.pyplot as plt
from urllib import request 

url = 'https://raw.githubusercontent.com/dsaint31x/OpenCV_Python_Tutorial/master/images/bad_rect.png'
fstr = 'bad_rect.png'
request.urlretrieve(url,fstr)
print('saved ok : bad_rect.png')

img0 = cv2.imread('./bad_rect.png')
img1 = img0.copy()

# к·ёл ҲмқҙмҠӨмјҖмқјкіј л°”мқҙл„ҲлҰ¬ мҠӨмјҖмқј ліҖнҷҳ
img_gray = cv2.cvtColor(img0, cv2.COLOR_BGR2GRAY) 
threshold, binary_img = cv2.threshold(img_gray, 127, 255, cv2.THRESH_BINARY)
plt.imshow(binary_img,cmap='gray')
plt.show()

# м»ЁнҲ¬м–ҙ м°ҫкё° 
contours, hierarchy = cv2.findContours(binary_img.copy(), 
                                      cv2.RETR_EXTERNAL,
                                      cv2.CHAIN_APPROX_SIMPLE)
contour = contours[0]


# м „мІҙ л‘ҳл Ҳмқҳ 5%лЎң мҳӨм°Ё лІ”мң„ м§Җм • 
app_rate = 0.05

#м „мІҙ contourл‘ҳл Ҳ
epsilon = app_rate * cv2.arcLength(contour, True) 
print('epsilon=',epsilon)

# к·јмӮ¬ м»ЁнҲ¬м–ҙ кі„мӮ° 
# мЈјм–ҙм§„ contour(кіЎм„  лҳҗлҠ” лӢӨк°Ғнҳ•)мқ„ epsilon(мҳӨм°ЁлІ”мң„)м—җ л§һм¶°
# contourм—җ мҶҚн•ҳлҠ” м җл“Өмқ„ мӨ„мқё approximation(к·јмӮ¬ м»ЁнҲ¬м–ҙ)лҘј л°ҳнҷҳ
#
# param 1 : target contour
# param 2 : мҳӨм°ЁлІ”мң„
# param 3 : contourк°Җ closeмқёк°Җ? True : closed
approx = cv2.approxPolyDP(contour, epsilon, True)

print('bf:{:.2f}| af:{:.2f}'.format(
    cv2.arcLength(contour, True), 
    cv2.arcLength(approx, True)
    )
)

# к°Ғк°Ғ м»ЁнҲ¬м–ҙ м„  к·ёлҰ¬кё° ---в‘Ј
cv2.drawContours(img0, [contour], -1, (0,255,0), 3)
cv2.drawContours(img1, [approx], -1, (0,255,0), 3)

plt.figure('original contour')
plt.imshow(img0[:,:,::-1])
plt.figure('approximated contour')
plt.imshow(img1[:,:,::-1])
plt.show()

    Convex Hull¶

    contourм—җ лҢҖн•ҙм„ң, н•ҙлӢ№ contourлҘј л‘ҳлҹ¬мӢёлҠ” лӢӨк°Ғнҳ•мқ„ Convex Hullмқҙлқјкі  л¶ҖлҰ„.

    convexity defectлқјлҠ” мҡ©м–ҙмҷҖ convexлқјлҠ” мҡ©м–ҙмқҳ к°ңл…җмқ„ кё°м–өн•  кІғ.

    OpenCVм—җм„ңлҠ” Sklansky algorithmмқ„ кө¬нҳ„н•ҳм—¬ cv2.convexHull()лЎң м ңкіөн•ңлӢӨ. 

    hull = cv2.convexHull(
    points,
    hull,
    clockwise=False,
    returnPoints=True
    • points : contourлҘј мқҙлЈЁлҠ” pointл“ӨлЎң кө¬м„ұлҗң listмһ„. мЈјмқҳн•  кұҙ, convexHullмқҖ н•ҳлӮҳмқҳ contourл§Ңмқ„ мһ…л ҘмңјлЎң л°ӣлҠ”лӢӨ. (ліөмҲҳк°ңмқҳ contourлҘј л„ҳкІЁмӨ„ мҲҳ м—ҶмқҢ)
    • hull : NoneмңјлЎң м§Җм •лҗҳлҠ” кІҪмҡ°к°Җ мқјл°ҳм Ғмқҙл©°, л°ҳнҷҳк°’мқҙ м ҖмһҘлҗң ліҖмҲҳлӘ…мқҙ л“Өм–ҙк°җ.
    • clockwise : orientationмңјлЎң л°ҳнҷҳлҗҳлҠ” convexHullмқ„ кө¬м„ұн•ҳлҠ” vertexл“Өмқҳ мҲңм„ңлҘј мӢңкі„л°©н–ҘмңјлЎң н• м§Җ л°ҳмӢңкі„л°©н–ҘмңјлЎң н• м§ҖлҘј кІ°м •.
    • returnPoints : Trueмқё кІҪмҡ°, convexHullмқ„ кө¬м„ұн•ҳлҠ” vertexл“Өмқҳ мўҢн‘ңл“ӨлЎң кө¬м„ұлҗң listлҘј л°ҳнҷҳн•ҳкі , Falseмқё кІҪмҡ°, мһ…л Ҙ argumentлЎң л“Өм–ҙмҳЁ pointsм—җм„ң convexHullм—җ лҢҖмқ‘н•ҳлҠ” vertexл“Өмқҳ indexл“Өмқ„ л°ҳнҷҳн•Ё.

    нҢҢмҶҗмқҙ лҗң л¶Җн’Ҳм—җм„ң нҢҢмҶҗлҗң мң„м№ҳл“ұмқ„ м°ҫлҠ” кІҪмҡ°м—җлҠ” convexity defeatмқҳ мң„м№ҳлҘј м°ҫм•„м•ј н•ҳлҠ” кІҪмҡ°к°Җ л§ҺлӢӨ. мқҙ кІҪмҡ°м—җлҠ” returnPointsлҘј FalseлЎң л„ҳкІЁмЈјм–ҙм„ң contour мӨ‘м—җм„ң м–ҙл–Ө indexмқҳ vertexк°Җ convexHullм—җ мҶҚн•ҳлҠ”м§ҖлҘј м°ҫмқҖ нӣ„, мқҙлҘј cv2.convexityDefects()м—җ contourмҷҖ н•Ёк»ҳ л„ҳкІЁмЈјм–ҙ м°ҫмқ„ мҲҳ мһҲмқҢ.

    Convexity Defects¶

    cv2.convexityDefects()лҘј нҶөн•ҙ м°ҫмқ„ мҲҳ мһҲмқҢ.

    • л°ҳл“ңмӢң convexHullмқ„ кө¬н•  л•Ң, retrunPoints=FalseлЎң мЈјкі  кө¬н•ҙм•јн•Ё.
    hull = cv.convexHull(cnt,returnPoints = False)
defects = cv.convexityDefects(cnt,hull)

    л°ҳнҷҳлҗң defectsлҠ” 4к°ңмқҳ elementлҘј к°Җм§ҖлҠ” vectorл“Өмқҳ listмһ„.

    к°Ғ rowм—җ н•ҙлӢ№н•ҳлҠ” vectorл“ӨмқҖ лӢӨмқҢмқҳ м •ліҙлЎң кө¬м„ұлҗЁ.

    • start point : contourм—җм„ңмқҳ indexм—җ н•ҙлӢ№н•Ё. convex hullм—җм„ңмқҳ мӢңмһ‘м җ.
    • end point : contourм—җм„ңмқҳ indexм—җ н•ҙлӢ№н•Ё. convex hullм—җм„ңмқҳ лҒқм җ.
    • farthest point : startмҷҖ endмӮ¬мқҙм—җ мһҲлҠ” convexity defectмқҳ index (contourм—җм„ңмқҳ index)
    • approximate distance to farthest point.

    example¶

    лӢӨмқҢ мҳҲм ңлҠ” convex hullмқ„ мқҙлЈЁлҠ” м җл“ӨмқҖ мҙҲлЎқмғү м„ мңјлЎң мқҙм–ҙм„ң лӢӨк°Ғнҳ•мқ„ л§Ңл“Өкі , convexity defeatм—җлҠ” л¶үмқҖмғүмқҳ мӣҗмңјлЎң н‘ңмӢңлҘј н–ҲмқҢ.

    import requests
import cv2
import numpy as np
import matplotlib.pyplot as plt

#os.makedirs('./tmp',exist_ok=True)

plt.imshow(tmp[...,::-1])
plt.xticks([]),plt.yticks([])
plt.show()
