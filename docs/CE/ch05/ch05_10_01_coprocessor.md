# Coprocessors

## Coprocessor란?

* Processor Core의 부담을 줄이기 위한 보조적인 H/W. 
* 일반적인 연산에 Core가 보다 집중적으로 할당되도록, 
* 단순하고 공통된 작업들을 더 단순한 하드웨어인 보조 프로세서에 위임: data copy 등
