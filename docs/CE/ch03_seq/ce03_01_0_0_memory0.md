# 시간 및 기억소자들

combinatorial logic과 달리, sequential logic의 경우 과거의 출력값에도 영향을 받기 때문에 memory device가 필요하며, 현재의 input 이외의 과거 상태 및 현재 상태 값들을 고려하여 출력이 결정되므로 time을 나타내는 device가 필요하다.
