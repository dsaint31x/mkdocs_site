# QtWidgets 모듈의 다양한 Widget Class들

이 장에서는 PySide6 및 PyQt6 에서 제공하는 다양한 Widget Class를 소개한다.

> `QtWidgets` 모듈은 GUI를 구축하기 위한 주요 Widgets 및 기능을 제공함. 

이 모듈이 제공하는 Widget 클래스들은 window, button, text box, layout 등과 같은 다양한 GUI 요소(widget)들을 만들고 조작하는 데 사용됨.  

다음의 Widget 클래스들은 앞서 살펴본 것이므로 이 장에선 다루진 않음:

* [`QLabel`](./ch02_s03_qlabel) : GUI에서 고정된 text 문자열 또는 image를 표시하는 Widget.
* [`QPushButton`](./ch02_s06_qpushbutton) :  GUI에서의 푸쉬 버튼을 abstraction하고 있는 Widget
